# Notas Técnicas e Referências de Cibersegurança

Bem-vindo à minha secção de notas estruturadas. Este espaço é dedicado ao armazenamento de conceitos teóricos, glossários de ferramentas e folhas de consulta (*cheat sheets*) que utilizo no meu percurso de aprendizagem em Red Team e Segurança Ofensiva.

O objetivo é manter uma base de conhecimento sólida, direta ao ponto e de consulta rápida para o dia a dia.
